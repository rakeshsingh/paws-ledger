## User Experience Flow 
### All Screens (Header)
* Have following Menu items: Home, Dashboard, About, and Login ( in that order)
* About menu should have following submenu: About, Contact, and FAQ
### All Screens (Footer)
### Screen 1: Landing page
* A visitor should be able to search for a missing pet's information using a microchip id 
* If the pet information is present on this website, the visitors should be asked to login, to see the details, and nudge the owner
* If the pet information is not present, do a lookup to AAHA website, and share details from the AAHA website, and nudge user to put the information on this website. 
* Landing page should have a button to nudge a visitor to login and resgister their pet.
